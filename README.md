# MIREA Java генератор отчётов с GitHub

### `Установка`

```
$ git clone https://github.com/m1cron/mireaPDF
$ cd mireaPDF
$ mvn package
```

### `Использование`

```
$ java -jar mireaPDF*.jar
```
